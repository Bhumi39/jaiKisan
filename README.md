# BHUMIIQ PRO - Enterprise Sustainable Agri-Intelligence

BHUMIIQ is a state-of-the-art agricultural intelligence portal designed for rural farmers and enterprise agri-consultants. It combines **Gemini 2.5 Flash** vision models with real-time weather and market discovery to provide actionable, bilingual roadmaps for crop success.

## 🚀 Vision & Mission
Our mission is to empower farmers with precision agriculture tools that are simple, accessible, and high-resolution. From detecting a leaf pathogen to predicting the next month's Mandi trends, BHUMIIQ is the "Brain" of the modern farm.

---

## ⚡ Core Intelligence Features
1.  **Crop Health Scanner & Reporter**: Vision-powered analysis with 10-item fertilizer roadmaps and "Intelligence Flowcharts".
2.  **Live Mandi Prices**: point-to-point statement cards with trend-aware market highlights.
3.  **AgroWeather & GPS Q&A**: Hyperlocal weather with an AI assistant to answer questions like "Can I spray tonight?".
4.  **Farm Pro Toolbox**: Specialized calculators for Irrigation, Yield, Cost, and Fertilizer (N-P-K).
5.  **Multilingual Voice Assistant**: Context-aware support in Hindi, Punjabi, Marathi, Bengali, and more.

## 🛠️ Setup & Installation
1.  **Clone the Repository**:
    ```bash
    git clone [your-repo-url]
    cd Bhumi39/jaiKisan
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Environment Setup**:
    - Copy `.env.example` to `.env`.
    - Insert your `GEMINI_API_KEY` from [Google AI Studio](https://aistudio.google.com/).
4.  **Run Locally**:
    ```bash
    npm run dev
    ```
